# Multi-Client Chat Application

A real-time chat application built using Java with client-server architecture.

## Features
- Multi-client support using multithreading
- Real-time messaging using sockets
- MySQL database integration
- GUI built with Java Swing

## Tech Stack
- Java
- Socket Programming
- MySQL
- Swing

## How to Run

1. Start MySQL and run schema.sql
2. Run ChatServer.java
3. Run ChatClient.java (multiple instances)
